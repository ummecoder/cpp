# C++ Complete Beginner Programs 💻
### By ummecoder

Ye repository C++ ke basic to intermediate concepts ke programs ke liye hai.  
Har concept ke 1-1 ya zyada examples diye gaye hain taki practice ho jaye.

---

## 📌 Topics Covered

### **1. Input / Output**
Basic `cin` aur `cout` ke programs
- User se data lena aur screen pe print karna

### **2. Arithmetic Operators**
- **Add** - 2 numbers ka sum
- **Minus** - Subtraction
- **Multiply** - Multiplication  
- **Division** - Divide

### **3. Conditional Statements**
- **if** - 1 condition check karna
- **if else** - 2 options
- **if else if** - Multiple conditions
- **switch** - Menu ya choice based programs

### **4. Loops**
- **for loop** - Fixed bar repeat karna
- **while loop** - Condition true hone tak repeat
- **do while loop** - Kam se kam 1 bar to chalay ga hi

### **5. Nested Loops**
- **Patterns** - Star, Number, Triangle patterns
- **Tables** - Multiplication table 1 to 10
